# Events Folder

This folder contains images related to CSCE events and ceremonies.

## Image Files:

### previous-unveiling.jpg
- **Usage**: "Next Plaque Unveiling" section on homepage
- **Content**: Photo from a previous plaque unveiling ceremony
- **Dimensions**: Recommended 600x400px
- **Format**: JPG
- **Description**: Shows dignitaries, attendees, or the actual plaque unveiling moment

## Suggested Image Content:
- Officials unveiling a historic site plaque
- Crowd gathered for ceremony
- CSCE representatives at the event
- The historic site with the new plaque
- Formal ceremony setting

## Usage in Code:
```html
<img src="images/events/previous-unveiling.jpg" 
     class="img-fluid rounded-3 shadow-lg" 
     alt="Previous Plaque Unveiling Ceremony">
```

The image will have an overlay caption that can be customized in the JavaScript file.
