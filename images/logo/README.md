# Logo Folder

This folder contains the CSCE logos used throughout the website.

## Logo Files:

### crest.png
- **Usage**: Top navigation bar
- **Size**: Small, compact logo (crest only)
- **Dimensions**: Recommended 120x120px or similar square format
- **Format**: PNG with transparent background

### horizontal.png
- **Usage**: Footer center
- **Size**: Horizontal layout with text and logo
- **Dimensions**: Recommended 300x100px or similar horizontal format
- **Format**: PNG with transparent background

## Usage in Code:

**Navigation Bar:**
```html
<img src="images/logo/crest.png" alt="CSCE Logo" height="50" class="me-3 logo-img">
```

**Footer:**
```html
<img src="images/logo/horizontal.png" alt="CSCE Logo" height="50" class="footer-logo">
```

## Design Guidelines:
- Use transparent backgrounds for both logos
- Maintain CSCE brand colors
- Ensure logos are crisp at specified sizes
- Test visibility on both light and dark backgrounds
